
1. @State修饰的响应式变量
@State是ArkUI核心响应式标记，被它修饰的变量一旦数值发生变化，页面上对应的显示内容会自动刷新，不用手动更新界面。
hour、minute、second：分别代表倒计时的小时、分钟、秒，初始值设置为0、0、30，程序打开默认显示00:00:30。
isRunning：布尔类型状态标记，用来记录计时器当前是否正在倒计时。运行时值为true，暂停时值为false，用来切换按钮上“开始/暂停”文字。
（1）启停按钮
按钮文字通过三元表达式动态变化：计时器运行时显示“暂停”，未运行时显示“开始”；按钮绑定点击事件，点击后判断当前运行状态，分别调用暂停、启动函数；自定义蓝色背景、白色文字，设置固定宽高。
（2）重置按钮
固定显示文字“重置”，点击直接执行重置函数；红色背景区分启停按钮，点击后计时器立刻停止，时间恢复初始30秒。
五、自定义功能函数文字讲解
1. formatNum 数字格式化工具函数
入参是数字，返回字符串。逻辑判断：如果数字小于10，就在数字前面拼接字符0，其余数字直接转为字符串。核心用途统一时间格式，例如数字5转为“05”，保证界面时间显示整齐规范。
2. startTimer 启动计时器函数
第一行判断：如果isRunning为true（计时器正在运行），直接结束函数，避免重复创建多个定时器造成程序卡顿；
修改运行状态isRunning为true，页面按钮文字同步切换为“暂停”；
setInterval创建循环定时器，每1000毫秒（1秒）自动执行一次内部代码；
倒计时逻辑：每秒秒数减1；
秒数小于0时，秒数重置为59，分钟减1；
分钟小于0时，分钟重置为59，小时减1；
小时小于0代表倒计时全部结束，自动调用暂停函数，并将时分秒全部清零。
3. pauseTimer 暂停计时器函数
将运行状态isRunning改为false，按钮文字切换回“开始”；
通过timerId调用clearInterval关闭循环定时器，停止倒计时；
将timerId赋值为空，释放系统定时资源。
4. resetTimer 重置计时器函数
先调用暂停函数停止当前正在运行的倒计时，再把时分秒三个变量恢复初始值0、0、30，界面时间同步回到00:00:30。
六、页面生命周期函数 aboutToDisappear 说明
该函数属于页面生命周期回调，作用是页面关闭、退出、跳转页面时自动执行。函数内部调用暂停计时器方法，强制销毁定时器。
如果不写这个函数，页面关闭后定时器会在后台持续运行，持续占用系统资源，造成内存泄漏、手机卡顿，是程序优化的关键代码。
七、整体运行逻辑文字总结
程序打开，页面加载，默认显示00:00:30，按钮文字为“开始”；
点击开始按钮，定时器每秒减少1秒，界面数字实时更新，按钮文字切换为暂停；
计时过程点击暂停，定时任务终止，数字停止变化；再次点击恢复倒计时；
任意时刻点击重置，计时停止，时间回到初始30秒；
倒计时全部走完（00:00:00），程序自动停止计时；
退出页面时，后台定时器自动销毁，不会残留占用资源。
